# BTBNS Documentation

## English

Welcome to the BTBNS documentation for bt_api.

### Quick Start

```bash
pip install bt_api_btbns
```

```python
from bt_api_btbns import BtbnsApi
feed = BtbnsApi(api_key="your_key", secret="your_secret")
ticker = feed.get_ticker("BTCUSDT")
```
